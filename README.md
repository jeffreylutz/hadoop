Installation
-------------
1. Clone the https://github.com/jeffreylutz/hadoop
2. Download virtualbox image:  http://.../hadoop.box
3. Run `vagrant box add 'hadoop.box' 'hadoop'`
4. Run `vagrant up`  --> You should have four VMs running in VirtualBox
